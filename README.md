# Boule de Poils 76

Site de présentation de l’association Boule de Poils 76

## Démarrage

1. Installez les dépendances :
   ```bash
   npm install
   ```
2. Lancez le serveur :
   ```bash
   npm start
   ```
3. Rendez-vous sur [http://localhost:5000](http://localhost:5000)

## À propos

Association à but non lucratif (loi 1901) dédiée à la protection des chats errants.

- Fondée par Maureille Marie-José
- Plus de 7 ans d’engagement
- Tous les dons et frais sont réinvestis pour le bien-être des chats

## Personnalisation

- Le style principal utilise la couleur #898ace.
- Modifiez le contenu dans `public/index.html` et le style dans `public/style.css`.
